Technologies

The places have been automatically extracted using Stanford and Freeling, Named Entity Recognizers, and geolocated with the R package editio/georeference, develop for this project. The final data has been refined manually.

The georeference and tiling of historical maps used QGIS. They are hosted in a github repository editio/tiles and served through URLs in the XYZ source format (zoom level and tile grid coordinates).

This site, the maps, the statistics and visualizations are build with R within the R Studio environment, using mostly the packages R Markdown and Leaflet for R.
