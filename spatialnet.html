A spatial feature of the genre models —Heliodorus’ Ethiopica, Cervantes’ Persiles— is the presence of a geographical division between core and periphery. This is an atempt to automatically compute the geographical periphery and center of the places mentioned in a text using network metrics (eigenvector values).

Some places have been refined again manually, e.g., locations like Thule has been added in Persiles.
